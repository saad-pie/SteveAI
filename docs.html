Introduction

Welcome to the official SteveAI Documentation. Learn how to chat with SteveAI, use commands, and integrate it into your workflow.

Getting Started

  1. Go to the Chat page.
  2. Type your question or request.
  3. Press Enter or click Send.

Features

  • 💬 Smart AI chat
  • 🌙 Dark/Light mode
  • 🗑️ Clear chat history
  • 📝 Markdown rendering

Commands

  • /clear – Clears chat history
  • /theme – Toggles theme
  • /help – Shows available commands

FAQ

Is SteveAI free?

Yes, SteveAI is free to use.

Does it save chats?

No, chats are temporary and not permanently stored.
